#  Calculadora de IMC

## 📖 Sobre o Projeto

Este é um projeto simples de **Calculadora de IMC (Índice de Massa Corporal)** desenvolvido com **HTML5, CSS3, JavaScript e Bootstrap**.

O principal objetivo foi praticar conceitos fundamentais de **lógica de programação com JavaScript**, trabalhando com entrada de dados, validações, cálculos matemáticos e exibição dinâmica de resultados na tela.

Além disso, o **Bootstrap** foi utilizado para criar uma interface responsiva e agradável, adaptando-se a diferentes tamanhos de tela.

Este projeto foi desenvolvido para fins de estudo e para acompanhar minha evolução na área de **Desenvolvimento Front-End**.

---

## Funcionalidades

* Inserção do nome do usuário;
* Inserção de peso e altura;
* Cálculo automático do IMC;
* Exibição da classificação correspondente ao resultado;
* Validação de campos obrigatórios;
* Feedback visual para resultados e mensagens de erro;
* Interface responsiva utilizando Bootstrap.

---

##  Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 4
* Font Awesome

---

##  Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados os seguintes conceitos:

* Manipulação do DOM;
* Eventos com JavaScript;
* Funções;
* Estruturas condicionais (`if`, `else if` e `else`);
* Validação de dados;

* Template Strings;
